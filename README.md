# YosemiForge
Fontforge Theme

##Install
###Mac
Just run InstallYosemiForge - note that this will overwrite the default FontForge graphics
###Manually
Copy the content of the Pixmaps folder to FontForge.app/Contents/Resources/opt/local/share/fontforge/pixmaps
